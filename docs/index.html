Rubik OS

Optimizada para memoria. Arquitectura descentralizada. Inspirada en el Cubo Rubik.

Descargar ISO

Última versión estable — v1.0.0 (2026-06-27)

ISO booteable UEFI. ~2.9 GB (comprimido: 2.8 GB en 2 partes).

Parte 1 (1.9 GB) Parte 2 (918 MB) Ver Release
Reconstruir la ISO 
cat rubik-os-*.iso.xz.part_* > rubik-os-20260627-x86_64.iso.xz
xz -d rubik-os-20260627-x86_64.iso.xz

Grabar en USB

Linux 
dd if=rubik-os-20260627-x86_64.iso of=/dev/sdX bs=4M status=progress && sync
Windows (Rufus)

Seleccionar la ISO → Modo: "DD Image" (NO "ISO Image") → Escribir

macOS 
sudo dd if=rubik-os-20260627-x86_64.iso of=/dev/diskX bs=4m status=progress

Repositorio de paquetes [rubik]

Agregá el repo oficial de Rubik OS a tu /etc/pacman.conf:

[rubik]
Server = https://erac73.github.io/rubik-os/repo/$arch

Paquetes disponibles:

  • rubik-core — Sistema base de Rubik OS (orquestador, celdas, caras, systemd units, perfiles AppArmor)
  • rubik-face-0 a rubik-face-5 — Scripts de cada cara del Cubo Rubik

Luego instalá con pacman:

pacman -Sy rubik-core

Acerca de

Rubik OS es una distribución Linux basada en Arch que funciona con 80-120 MB RAM en idle. Reorganiza cada componente como una celda independiente — como las caras de un Cubo Rubik — con su propio cgroup, perfil AppArmor y límites de recursos.

80-120 MB
RAM idle (sin GUI)
~80-150
Procesos en idle
56
Celdas atómicas
6
Caras (F0-F5)

Ideal para laptops viejas, single-board computers, VMs y cualquiera que quiera un Linux modular y eficiente.
