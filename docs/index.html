About Rubik OS

Acerca de Rubik OS

Rubik OS reorganizes the operating system into 58 atomic cells arranged across 6 faces (F0-F5), like a Rubik's Cube. Each cell has its own cgroup, AppArmor profile, and resource limits. If one cell fails, the rest keep running.

Rubik OS reorganiza el sistema operativo en 58 celdas atómicas distribuidas en 6 caras (F0-F5), como un Cubo Rubik. Cada celda tiene su propio cgroup, perfil AppArmor y límites de recursos. Si una celda falla, las demás continúan.
