Quick Setup

Configuración Rápida

Add the official Rubik OS repository to your /etc/pacman.conf:

Agregá el repositorio oficial de Rubik OS a tu /etc/pacman.conf:

[rubik]
Server = https://erac73.github.io/rubik-os/repo/$arch

Then update and install:

Luego actualizá e instalá:

pacman -Sy rubik-core

Build from Source

Construir desde Fuente

Clone the repository and build the ISO using Docker:

Cloná el repositorio y construí la ISO usando Docker:

git clone https://github.com/erac73/rubik-os.git
cd rubik-os
./scripts/docker-build.sh

The script builds the full ISO inside a Docker container using archiso.

El script construye la ISO completa dentro de un contenedor Docker usando archiso.

Build Packages Individually

Construir Paquetes Individualmente

Each package can be built with makepkg:

Cada paquete puede construirse con makepkg:

# Build rubik-core# Construir rubik-core
cd packages/core/rubik-core
makepkg -si

# Build a face package# Construir un paquete de cara
cd packages/faces/face-0
makepkg -si
