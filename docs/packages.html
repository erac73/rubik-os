Setup

Configuración

Add the official Rubik OS repository to /etc/pacman.conf:

Agregá el repositorio oficial de Rubik OS a /etc/pacman.conf:

[rubik]
Server = https://erac73.github.io/rubik-os/repo/$arch

Then update and install:

Luego actualizá e instalá:

pacman -Sy rubik-core

Build from Source

Construir desde Fuente

git clone https://github.com/erac73/rubik-os.git
cd rubik-os
./scripts/docker-build.sh

Or build individual packages with makepkg:

O construir paquetes individuales con makepkg:

cd packages/core/rubik-core
makepkg -si
